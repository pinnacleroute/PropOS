# PropOS

> A modern **property management operating system** for landlords and real estate managers — streamlining listings, tenants, leases, and maintenance in one place.

🌐 **Live Demo:** [PropOS on GitHub Pages](https://pinnacleroute.github.io/PropOS)

---

## ✨ Features

- 🏢 Property & unit management
- 👤 Tenant profiles & lease tracking
- 💰 Rent collection & payment history
- 🔧 Maintenance request tracking
- 📊 Occupancy & revenue analytics
- 📱 Fully responsive UI

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | HTML5, CSS3, JavaScript |
| Deployment | GitHub Pages |
